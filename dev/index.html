TechnicalIndicatorCharts.abbrev — Method
abbrev(p::Period)

Return an abbreviated string representation of the given period.

Example

abbrev(Hour(4)) # "4h"
abbrev(Day(1))  # "1d"

TechnicalIndicatorCharts.chart — Method
  chart(name, tf; indicators, visuals)

Construct a Chart instance configured with the given indicators and visual parameters.

Example

julia> golden_cross = chart(
    "BTCUSD", Hour(4);
    indicators = [
        SMA{Float64}(;period=50),
        SMA{Float64}(;period=200)
    ],
    visuals = [
        Dict(
            :label_name => "SMA 50",
            :line_color => "#E072A4",
            :line_width => 2
        ),
        Dict(
            :label_name => "SMA 200",
            :line_color => "#3D3B8E",
            :line_width => 5
        )
    ]
)

TechnicalIndicatorCharts.merge_candle! — Method
merge_candle!(last_candle, c)

If last candle is not provided, construct a new candle with the given OHLCV data. If last candle is provided, mutate lastcandle such that it's HLCV are updated. It's assumed that lastcandle and c have the same timestamp.
